Mindomo


Mindomo és una eina web i mòbil per la creació de mapes mentals i conceptuals de forma gratuïta i senzilla en linea.

Permet introdiur tota classe de contingut, ja sigui text, enllaços, vídeos, música o imatges.
A més tenim la possibilitat d'exportar el mapa en qualsevol dels següents formats

  • Imatge
  • Arxiu de text sense format
  • RTF (com Microsoft Word)
  • PDF


Funcionament


Aqui hi ha un mapa conceptual fet amb mindomo que t'ensenya com funciona.

Per crear els mapes mentals has d'estar registrat, una vegada registrat pots fer fins a 3 mapes, ja que té dos versions la versió gratuïta i les versions pagant, que hi han diferents quotes, depenent del que necessitis.


Funcions i Prestacions


Aquesta app o pàgina web el que ens permet fer és crear mapes conceptuals o mentals per així organitzar-nos millor, o estudiar més fàcilment. Ja que et permet fer 3 mapes gratuïts el que pots fer és crear un de nou i quan en tinguis tres els borres i així podràs fer molts més mapes.
